Trapped!

Introduction

Trapped! is a 2D Java game I developped, the objective is to traverse the square area with keyboard keys while trying to avoid making contact with the opponent snake.

This game is based on the classic game Snake. In this version you can either play against an AI snake or you can play with a friend using different keyboard keys for movement. You can also change your snake colour and difficulty of AI.
