Warehouse

Introduction

Warehouse is a 2D Java game I developped, the objective is to move around boxes and to match boxes of the same colour.

Technology

Warehouse implements java threads, graphics, and action listeners. Graphics were handled by Java GUI and specific images were deisgned on Adobe Photoshop.

Different levels are stored in Text Files representing an 8x10 matrix. As an example level 1 looks like: 1111111111 1000000001 1020000201 1000000301 1000000001 1023000001 1000000001 1111111111

Where 1 are the walls, 0 is empty space, 2 is a blue box, 3 is a red box.
